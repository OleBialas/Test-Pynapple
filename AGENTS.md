The purpose of this project is to learn about the Pynapple Python package.
The IPYNB files in notebooks/ contain different projects to test its capabilities and limitations.

When answering questions about Pynapple, consult the official docs: https://pynapple.org/api.html and inspect the notebook state via Jupyter MCP wherever appropriate.

Code should ideally be simple and teachable. Only doe explicit transformations between different datatypes (numpy array, pandas df etc.) if Pynapple does not offer a ready solution.
